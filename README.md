# odin-rock-paper-scissors
컴퓨터가 랜덤하게 가위바위보중에서 고르는 펑션 만들기
인간이 가위바위보중에서 적어내는 프롬프트 만들기
두 선택을 비교해서 승패 결정해주는 알고리즘 만들기
양측 승점 표시기능 만들기

비기는 경우 == 둘이 같은 선택을 할 경우

getHumanChoice() --> 프롬프트 넣어서 사람 답변 추출
getComputerChoice() --> 난수 넣어서 컴퓨터 답변 추출
playRound() --> 두 답변 비교해서 승패 가리고 이긴 쪽 승점 1점 올려주는 기능
                                (승패 표시하는 답변 나오게 alert)
